# Dynamic Voice Response Manager

## Objective

Control speaking flow dynamically using Text To Speech.

## Features

- Text to Speech Conversion
- Play Voice
- Pause Voice
- Resume Voice
- Stop Voice
- Adjustable Speed
- Queue Management
- GUI Interface

## Technologies

- Python
- Tkinter
- pyttsx3
- Queue
- Threading

## Run Project

Install dependencies

pip install -r requirements.txt

Run

python main.py

## Testing

python test_voice_controller.py
